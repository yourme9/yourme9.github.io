html页面常用布局

OneWord 2020-11-11 html

# 一、两边固定，中间自适应

运用浮动  
中间主体部分不固定宽度，两侧部分分别使用浮动

# 二、flex弹性布局

# flex-dirction:控制子项整体布局反向.

row-----水平排列
row-reverse-----反向水平排列
column-----垂直排列
column-reverse-----反向垂直排列

# flex-wrap：控制子项单行显示还是换行显示。

nowrap-----单行显示，不换行
wrap----宽度不足换行
wrap-reverse----宽度不足换行显示 反向

# flew-flow :以上两个属性符合写法

第一个属性方向 第二个换行
dirction、wrap的复合写法

# justify-content设置主轴方向上子项对齐方式

flex-start----起始位置对齐
flex-end----结束位置对齐
center----居中对齐
space-between----左右对齐 中间平均分配
space-around----每个子项左右间距相等，平均分配
space-evenly----空间大小相同

# align-items设置侧轴上子元素排列方式(单行)

flex-start----顶部对齐
flex-end----底部对齐
center----居中对齐
streth----拉伸(默认值)

# align-content设置侧轴方向上子项对齐方式(多行)

flex-start----起始位置对齐
flex-end----结束位置对齐
center----居中对齐
space-between----左右对齐 中间平均分配
space-around----每个子项左右间距相等，平均分配
space-evenly----空间大小相同

# 子项上CSS属性

order----排列顺序 值越大越往后
flex-grow---- 1有空余位置时全部占用  0.5时占用百分之50  0不占用
flex-shrink----在空间不足时使元素收缩 0不收缩 1正常收缩
flex-basis----有剩余空间是给予一个宽度固定值
flex----上三个复合写法
align-self----单个子项垂直对齐方式 写法同align-items

# 三、移动端流式布局

使用百分比布局
通过盒子的宽度设置成百分比来根据屏幕的宽度来进行伸缩

# 四、移动端rem布局

通过rem单位来实现
1rem等于根元素所设置的字体大小
使用js动态改变html的字体大小
也可以使用vh，vm来设置根元素的字体大小
注意改变字体大小后要重置body字体大小

# 五、响应式布局

利用css媒体查询
分别设置不同屏幕尺寸下的样式
语法：@media screen and (max-width:767px){}
可使用bootstrap框架
